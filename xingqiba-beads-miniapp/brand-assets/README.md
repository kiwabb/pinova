## 字体合规记录

当前 Logo 来源是 AI 生成的位图，没有随附字体名称、字体文件或授权证明，因此无法从图片反向确认具体字体及其授权范围。项目没有嵌入或分发该图片背后的字体文件，Logo 在小程序中作为整体图片使用。

后续若需要重做可编辑的文字版本，辅助中文建议使用 Adobe 思源黑体，英文建议使用同为 SIL Open Font License 1.1 的字体，并保存所用版本及许可证副本。不要使用来源不明的商业字体文件重新排版。

- 思源黑体官方仓库与许可证：https://github.com/adobe-fonts/source-han-sans
- SIL Open Font License：https://openfontlicense.org/
