问题诊断 旧版左右分栏让短文案对高列表，行内文案又只贴左半边，出现两块「死白」；调间距治不了结构。
设计决策 抬头通栏；六格色样矩阵用拼豆阵列填满单元格；共享规则线、无悬浮阴影，符合 Color Index 禁止 pastel 卡墙的约束。
延伸到其他页 同类「左标签 + 右空宽内容」改为「通栏抬头 + 满宽矩阵/货架」；货架、创作路径、分类 masthead 同一密度语言。
